# PROJECT-DARK-ANGEL

Android APK Application with Login, Dashboard, and Bug Crash Features

## Features

- **Login Screen**: Username and password authentication
  - Username: `yan`
  - Password: `yan123`
- **Dashboard**: Post-login interface with multiple features
- **Bug Crash Module**: Select and attack various bug types
- **UI Design**: Black and red flashing background with sleek appearance

## Project Structure

```
app/
├── src/
│   ├── main/
│   │   ├── java/com/example/darkangel/

│   │   │   └── BugCrashActivity.kt

```

## Building the Project

1. Clone the repository
2. Open in Android Studio
3. Build and run on an Android device or emulator

## Dependencies

- Android SDK 33
- Kotlin
- AndroidX
